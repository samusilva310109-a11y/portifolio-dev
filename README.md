# Portifólio Dev
## Objetivo 🎯

Desenvolver um portifólio profissional, afim de promover a imagem do profissional e destacar os seus trabalhos e projetos. Trabalhar com a organização de projetos e elaboração do documento TAP (Termo de Abertura de Projeto)

## Tecnologias Utilizadas
* HTML - estrutração da página web
* CSS - estilização da página
* Javscript - interações e configurações do portifólio

## Estruturação do portifólio

1. Hero - Apresesntação breve do profissional e carreira desejada
2. Secção 1 - Stacks e habilidades (Front-end, Back-end, Banco de dados, etc)
3. Secção 2 - Cards com projetos e links de redirecionamento para os respectivos portifólios
4. Secção 3 - Cursos extracurriculares
5. Footer -  Rede de contato (Email, telefone, linkedin, github)
